Blocks

Pink2Web is using a block programming abstraction, which is easy to learn and provide excellent overview of functionality, even for non-programmers.

Start out with the large standard library of blocks, and expand your own library of re-usable blocks by grouping blocks to create larger blocks.

Block programs are always running, even while being modified. And with real-time feedback of values, you get a very interactive experience without waiting time for compilation and uploading to target system.

Collaboration

Pink2Web allows many people to modify the same program at the same time. Any changes made by Alice will be updated on Bob's browser immediately. Pink2Web also operates with a central sandbox for development independent of a target system, yet can push the new program to other systems with a single-click, and if someone is looking at or working on that target system, the new program will be visible immediately.

Pink2Hub is a public sharing hub, where you can publish your blocks for others to import and use.

Try!

Real-time

Pink2Web does not make a distinction between compile-time and runtime. Pink2Web programs are always running, even if they are empty. As blocks are added and wired to other blocks, the new configuration is immediately active.

Selecting one or more wires will display the current values on each of them. Selecting one or more blocks will display the values on each of its/their ports. Selections can be pinned or saved to be recalled at any time later.

Features

Block Programming

Block Programming is a superior paradigm for programming the IoT and automation systems. The clarity of the actual functionality of the running program is outstanding.

Live Programming

Not only does Pink2Web bring block programming to the IoT world and automation systems, but it does it with a real-time connection. Interactive development is faster, without edit-compile-test-debug cycles.

Debugging

Debugging is a breeze as one can easily view any value in the system at any point in time, during programming or while troubleshooting later.

Views of values-of-interest can be saved and used later. In many cases, these views can replace the need to develop costly HMIs (Human-Machine-Interface).

Collaboration

Never before has IoT and industrial automation programming been able provide this level of collaboration during development. Working on the same functionality at the same time without messy synchronization operations. Sharing your work with others. Search Pink2Hub for ready-to-use solutions. Organizations can have private repositories.

Small & Powerful

Pink2Web is a single, native executable of less than 5MB on disk, uses 1MB of RAM and 1000 blocks uses approx another 1MB of RAM.

Pink2Web is available for both x64 and ARM Linux.
